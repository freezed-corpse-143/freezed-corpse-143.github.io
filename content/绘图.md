# 时序图

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端页面
    participant B as 后端服务器
    participant D as 数据库

    U->>F: 点击登录按钮
    F->>B: 发送用户名和密码
    B->>D: 查询用户信息
    D-->>B: 返回用户数据
    alt 验证成功
        B-->>F: 返回 token 和用户信息
        F-->>U: 显示登录成功，跳转首页
    else 验证失败
        B-->>F: 返回错误信息（用户名或密码错误）
        F-->>U: 显示错误提示
    end
```
